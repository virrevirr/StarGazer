# StarGazer our project
Short description of our project

Our project is for the people that love nature and love watching the stars.
Our app tells you, given a city, what the weather, moonface as well as when the sunrise and sunset is so you can enjoy stargazing as long as possible. Another fun fact! is that it also gives you some details about different natural events occurring in the location you are at.
If it is that you lOOOVE the place you can also add it to your profile and save it there forever!
