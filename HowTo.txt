En liten how to guide om någon glömmer bort hur man kan se appen i realtid
I terminalen när man öppnar VS code pånytt:
--> npm run dev
--> öppna local host länken m. ctrl + click. (eller command + click för apple)

För att "användarna" ska se ändringarna måste man i terminalen skriva:
--> firebase deploy'

Vill du titta på en specifik presenter i appen?
gå in i index.jsx filen som ligger i src, och gör som exemplet nedan. Viktigt att det är en 
punkt i path innan första "/". Glöm inte att ändra tillbaka till ursprunget innan du pushar till 
main. 

import Information from "./presenter/informationPresenter.jsx";

import {createApp} from "vue";
function App(){ return <div><Information/></div>; };
const app= createApp(App);
app.mount("#root");

--> clicka "save all" och om du gjort som högst upp så kan du nu se presentern i local host.
